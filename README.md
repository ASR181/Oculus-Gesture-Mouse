# Oculus Gesture Mouse 👁️🖱️

A modern, AI-powered desktop application that allows you to control your computer mouse using hand gestures. Built with Python, OpenCV, MediaPipe, and PyQt6.

## ✨ Features
- **Cursor Movement:** Move your index finger to control the mouse precisely.
- **Clicking:** Pinch your index and middle finger together to left-click.
- **Smart Edge Scrolling:** Move your hand to the top or bottom of the screen to scroll web pages.
- **Modern UI:** Features a sleek, responsive PyQt6 interface with Light/Dark mode themes.
- **Calibration:** Real-time sliders to adjust cursor smoothness and movement boundaries.
