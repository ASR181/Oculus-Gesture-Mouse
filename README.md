# Oculus Gesture Mouse 👁️🖱️

A modern, AI-powered desktop application that allows you to control your computer mouse using hand gestures. Built with Python, OpenCV, MediaPipe, and PyQt6.

## 🚀 Download & Run (No Python Required)
If you just want to use the application without touching any code, you can download the ready-to-use version!

1. Go to the [Releases Page](https://github.com/ASR181/Oculus-Gesture-Mouse/releases/latest).
2. Download the `Oculus-Gesture-Mouse-Windows.zip` file under **Assets**.
3. Extract (unzip) the folder anywhere on your PC.
4. Open the folder and double-click **`main.exe`** to start the app!

*(Note: Your antivirus might flag the `.exe` as suspicious because it is a new, unsigned application. This is a normal "false positive" for Python desktop apps. You can safely select "Run anyway" or add it to your exclusions.)*

---

## ✨ Features
- **Cursor Movement:** Move your index finger to control the mouse smoothly and precisely.
- **Left-Clicking:** Pinch your index and middle finger together to click.
- **Smart Edge Scrolling:** Move your hand to the top or bottom of the screen to effortlessly scroll web pages.
- **Modern UI:** Features a sleek, responsive PyQt6 interface with Light ☀️ and Dark 🌙 mode themes.
- **Live Calibration:** Real-time sliders to adjust cursor smoothness and frame reduction boundaries.
